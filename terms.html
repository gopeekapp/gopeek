2. Description of Service

GoPeek is a browser extension that allows users to preview web links in interactive mini browser windows without leaving the current page. The Extension is available as a free version with limited features and a paid Pro version with additional functionality.

3. Free Plan

The free version of GoPeek includes:

  • Unlimited basic link previews
  • Sidebar and bubble minimize features
  • 2 custom URL navigations per day
  • 1 multi-peek session per tab per day

4. Pro Plan

The Pro plan is a paid subscription that unlocks:

  • Unlimited custom URL navigations
  • Unlimited multi-peek sessions
  • Ghost mode for pinned windows
  • Search selection feature

Pro subscriptions are billed through Paddle and are subject to Paddle's terms and conditions.
